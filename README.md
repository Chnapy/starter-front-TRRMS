# starter-front-TRRMS

Boilerplate **frontend** composed of:
- **Typescript**
- **React**, with `create-react-app`
- **Redux**, with `@reduxjs/toolkit`
- **Material-UI**
- **Storybook**

Also note use of:
- **Jest**, for testing
- **Github Actions**, as CI

Start with
```
yarn start
```

Test with
```
yarn test
```

Run Storybook with
```
yarn storybook
```
